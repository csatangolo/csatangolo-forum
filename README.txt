A regisztrációs oldal linkje: regisztracio.html

Apps Script beállítás:
1. Hozz létre egy Google Táblázatot.
2. Bővítmények -> Apps Script.
3. Illeszd be a Code.gs tartalmát.
4. A SPREADSHEET_ID értékét cseréld ki a Google Táblázat ID-jára.
5. Deploy -> New deployment -> Web app.
6. Execute as: Me.
7. Who has access: Anyone.
8. A kapott Web App URL-t másold be a regisztracio.html fájlban ide:
IDE_JON_A_GOOGLE_APPS_SCRIPT_WEBAPP_URL
